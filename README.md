# Resume_Praser_Navtech
Navtech Resume Praser Assignment
AI-powered Resume Parser that extracts structured data from PDF/DOCX resumes using LLMs.  
Built with Python, LangChain, and Groq for fast and accurate entity extraction.  
Outputs standardized JSON including contact info, education, experience, and skills.  
Easy to extend, integrate, and deploy as an API or automation tool.
